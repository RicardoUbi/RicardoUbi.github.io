Iniciando na programação - Um guia para iniciantes

Neste artigo, vou compartilhar minha jornada de aprendizado com programação. Abordarei os conceitos básicos, as principais dificuldades que enfrentei e as ferramentas que utilizei para estudar e desenvolver programas.

O inicio

Desde pequeno, sempre tive curiosidade sobre a tecnologia, mas foi uma conversa de aula com um professor que me empurrou nesse caminho. Lembro que estavámos conversando sobre carreira e sucesso profisional, e ele sugeriu que investissemos em certos tópicos que certamente nos trariam um diferencial que poderia abrir inúmeras portas, e entre esses tópicos estava a programação. A ideia parecia interessante, mas eu não tinha uma noção clara do que esperar. Para mim, assim como para muitos que não estã familiarizados, a programação parecia algo complexo, usado por hackers para invadir qualquer sistema com poucos comandos no teclado.

Foi então que, por coincidencia, encontrei uma disciplina optativa chamada "Algoritmos e Lógica de Programação" e decidi me inscrever sem muita pretensão. Quando comecei as aulas começaram e a professora explicou do que se tratava a materia, a primeira coisa que veio na minha mente foi: E agora? Por onde começar?

Por onde Começar?

Foi nesse momento que busquei a ajuda de mentores e professores. E as respostas eram bem semelhantes entre si: Comece pela base, uma base forte vai te permitir aprender mais rápido e resolver qualquer problema. A minha recomendação é começar com uma linguagem de programação que seja fácil para você, mas como você poderia saber, não é mesmo? Só existe uma resposta, experimentando. Testar diferentes linguagens e ferramentas é a unica forma de descobrir o que funciona melhor para você.

Então, primeiramente aprenda a usar um editor de código. Isso é essencial para começar a escrever suas primeiras linhas de Programação. Depois disso, o proximo passo é escolher uma linguagem.

Escolhendo a Primeira Linguagem

Para muitos iniciantes, Python pode ser uma excelente escolha. Sua sintaxe é simples e legivel, o que te permite focar no aprendizado da lógica de programação, em vez de se perder em detalhes técnicos. Além disso, Python é extremamante versátil, sendo usado em áreas como desenvolviemnto web, análise de dados, IA, automação, dentre outras.

Outras boas opções para iniciantes:

  • JavaScript: Afim de conhecer um pouco sobre desenvolvimento web, JavaScript é uma excelente escolha. Ele é essencial para criar sites dinâmicos e interativos.
  • HTML e CSS: Não são linguagens de programação, mas são fundamentais para quem deseja trabalhar com a parte visual do sistema (front-end). Com eles, você aprende a estruturar e estilizar páginas web.
  • Scratch: Conheci essa linguagem em um curso de ciência da computação que fiz, ela é ótima para quem nunca programou antes, especialmente crianças ou pessoas sem nenhum contato com tecnologia. Ele ensina lógica de programação de forma visual, com blocos que você arrasta e solta para criar seus próprios projetos.

Minhas escolhas

No meu caso, comecei aprendendo Java e Python ao mesmo tempo. De alguma forma, essas linguagens se complementavam de uma fomra interessante na minha cabeça. Enquanto Python me ajudava a compreender a lógica de uma forma intuitiva, Java me ajudava a entender a importância da estrutura e como a maquina interpreta e executa o código.

Entendendo os Conceitos Básicos

Agora que você já escolheu sua linguagem, vou te mostrar os conceitos fundamentais da programação que você deve entender no início. Nesses exemplos estarei utilizando Python:

Variáveis, o que são?

Imagine uma caixa. Você pode colocar qualquer coisa dentro dela: um livro, um boneco, um mouse. Essa caixa, na programação, é uma variavel.

Na programação, uma variavel é como uma caixa onde você armazena dados. Esses dados podem ser de diversos tipos, palavras, numeros, valores verdadeiros ou falsos, dentre outros.


    # Exemplo em Python
    variavel = "Olá"

Tipos de Dados

Cada variavel vai armazenar um tipo de dado, inicialmente vamos focar em Strings, Inteiros, Floats, e Boleanos.

  • Strings: Sequências de caracteres, como letras, números e símbolos, utilizadas para representar textos. São usadas para armazenar nomes, endereços, mensagens ou qualquer outro tipo de texto.
  • Inteiros: Números inteiros, ou seja, números sem virgula. São usados para representar quantidades exatas, como idades, contagens e anos.
  • Floats: Números com virgula, utilizados para representar valores decimais. São utilizadas para representar medidas, preços, valores com casas decimais.
  • Boleanos: Valores lógicos que representam verdadeiro (true) ou falso (false). São usados para representar condições, como resultados de comparações ou estados.

Exemplos de Strings, Inteiros, Floats, e Boleanos em código.


    # Exemplo em Python
    string = "Olá mundo!"
    int = 60
    float = 3.75
    bolean = False

Estruturas de Controle

As estruturas de controle permitem que seu programa tome decisões e repita ações de acordo com as condições determinadas. Em outras palavras, tornam seu programa mais dinamico e inteligente. Inicialmente vamos falar de If e else, For, e While.

  • If e else: É como um cruzamento com uma placa de sinalização: Se uma condição for verdadeira, o programa segue por um caminho, caso contrario, segue por outro. A clausula "else" é usada para definir o que acontece quando a cndição "if" não é verdadeira.
  • For: É como um loop que repete um bloco de código um numero determinado de vezes. É ideal para fazer um contador, por exemplo.
  • While: É outro tipo de loop, mas que ao contrario do "For", ele continua se repetindo enquanto uma determinada condição for verdadeira. É util para quando você não sabe exatamente quantas vezes o loop será executado. 

    # Exemplo de estrutura condicional
    # (If e else)
    idade = 19
    if idade >= 18:
        print("Você é maior de idade.")
    else:
        print("Você é menor de idade.")


    
    # Exemplo de loop (For)
    for numero in range(5):
        print("Contador: ", numero)


    
    # Exemplo de loop (While)
    numero = 15
    while numero > 10:
        print("Regressivo: ", numero)
        numero -= 1

Qual estrutura usar?

  • If e else: Quando você precisa tomar uma decisão única.
  • For: Quando você sabe o número exato de repetições.
  • While: Quando você não sabe o número exato de repetições e precisa de uma condição para parar o loop.

Funções

Uma função é um bloco de código que realiza uma tarefa especifica e que possui um nome. É como uma receita: Você define os ingredientes (parâmetros) e os passos (ações da função) para preparar um determinado prato (resultado). É uma ótima opção para evitar repetir o mesmo código varias vezes e torna o programa mais fácil de entender e melhorar. 


    # Exemplo de função em Python
    def saudacao(nome):
        print(f"Olá, {nome}!")
    
    saudacao("João")

Neste exemplo acima, foi criado uma função e depois a mesma foi chamada pelo seu nome junto do ingrediente (parâmetro), e então retorna a saudação + nome.

Dificuldades

Aprender a programar não é um caminho sem obstáculos. É normal se sentir perdido ou frustrado, acredito que a grande maioria passa ou já passou por isso.

Se sentir Sobrecarregado com a quantidade de informações e conceitos novos é um sentimento comum. Programar envolve uma curva de aprendizado íngreme, e tentar absorver tudo de uma vez pode ser sufocante. Nesse caso, quebrar o aprendizado em pequenas partes e aprofundar gradualmente é o melhor caminho. E claro, reserve um tempo para descanso.

A comparação com outros pode ser uma armadilha. É importante lembrar que cada pessoa tem seu próprio ritmo de aprendizado. Foque no seu progresso: sempre celebre suas pequenas e grandes conquistas e evite se comparar com os outros.

A falta de prática é um obstaculo comum. A programação é uma habilidade que se desenvolve com a prática. Separe um tempo todos os dias para codar, a consistencia é a chave. Construa pequenos projetos, isso te ajudará a aplicar os conceitos que você aprendeu e a ver seus progressos na prática.

Ferramentas que podem facilitar seu aprendizado

Durante meu processo inicial de aprendizagem, conheci algumas ferramentas que me ajudaram bastante. Acredito que elas podem te ajudaram tanto quanto me ajudaram.

Plataformas de Aprendizado

  • FreeCodeCamp: Excelentes cursos para quem esta iniciando, e o melhor, tudo de graça.

Ambientes de Desenvolvimento

  • Jupyter Notebook: Ótimo para aprender Python, especialmente em ciência de dados.
  • Git e GitHub: Ferramentas essenciais para versionamento de código.
  • Replit: Ótimo ambiente de desenvolvimento online com suporte a diversas linguagens, ou seja, pode ser usado de qualquer lugar.

Conclusão

Aprender a programar é aprender a criar soluções, é entender o que é ser persistente e principalmente, paciente.

Explore, experimente e mantenha sua curiosidade sempre acesa. A tecnologia está em constante evolução, e a programação oferece infinitas possibilidades. Então, escolha sua primeira linguagem, construa pequenos projetos, enfrente todos os desafios e, acima de tudo, divirta-se!
